Учебный проект по свёрточным нейронным сетям. В ноутбуке convolutionalNets.ipynb примеры использования фреймворка tensorflow на примерах с возрастающей сложностью. Последняя модель для распознавания автомобилей привела к созданию оконного приложения, где также используется готовая модель для переноса стилей. 
Датасет для обчения модели классификации доступен по ссылке:
https://drive.google.com/drive/folders/1fF6UUqtMTTguQYX-dUeyBD4362tZoCFh?usp=sharing

---ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОГРАММЫ---
1) В качестве интерпретатора должен использоваться 3.7<=Python<3.11.

2) Убедитесь, что используется последняя версия pip. 
Для этого в терминал вставьте команду python -m pip install --upgrade pip

3) В файле requirements.txt содержатся версии библиотек, используемых в проекте. 
Среда разработки предложит установить эти библиотеки(если этого не произошло - используйте команду pip install -r requirements.txt), для выполнения программы сделать это необходимо.

4) По ссылке из файла model_url.txt загрузите веса для модели и поместите файл weigths.h5 в папку с проектом.

5) После успешной установки библиотек и загурзки весов программа будет готова к работе.
